Методы строк:

- `string.at(pos)` - получить элемент по индексу pos (`.at(-1)` - получить последний элемент)

- `string.toLowerCase()`
    
- `string.toUpperCase()`

- `string.indexOf(substr, pos)` - поиск подстроки

- `string.slice(start, [, end])` - получение подстроки 

- `string.substring(string2)` - получение подстроки

- методы `includes/startsWith/endsWith` - проверка вхождения

- `string.localeCompare(string2)` - **правильное сравнение строк**

- `string.trim()` - обрезка пробелов в начале и конце строки

### [Остальные методы можно найти в справочнике](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/String)
