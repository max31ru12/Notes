

## Растяжение картинки

### Базовое правило

```css
img {
	max-width: 100%;
	height: auto;
}
```

- картинка не выйдет за контейнер
- пропорции сохраняются
- адаптивно для mobile

### Фиксированный контейнер

```css
.image-wrapper {
	width: 300px;
	height: 200px;
}

.image-wrapper img {
	width: 100%;
	height: 100%;
	object-fit: cover;
}
```


#### object-fit

| Значение    | Что делает                     |
| ----------- | ------------------------------ |
| **cover**   | заполняет блок, обрезая лишнее |
| **contain** | целиком влезает, без обрезки   |
| **fill**    | растягивает                    |
