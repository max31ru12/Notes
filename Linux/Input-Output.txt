ЧТЕНИЕ ФАЙЛОВ:
cat			- чтение всего файла
nl 			- вывести весь файл с номерами строк (или cat -b)
head			- первые 10 строк
tail			- последние 10 строк (или можно указать кол-вл строк tail -4)	
less			- смотреть файл по чуть-чуть (или можно указать кол-вл строк head -4)


1 > file 		- вывод в файл
2 > /dev/null		- направлять ошибки в /dev/null (игнорировать)
& > file		- направлять stdout и stderr в файл
