ПУТИ В ЛИНУКСЕ
~/ 	- домашняя директория пользователя
./ 	- текущая директория
../ 	- директория на уровень выше
/ 	- корневой каталог


ПАПКИ:
/usr/local/bin 		- программы пользователя
/usr/bin		- команды пользователя
/bin			- основные команды ОС


ФАЙЛЫ:

ССЫЛКИ:

Символические ссылки:
1) имею свои права доступа
2) разный inode с ориг. файлом
3) не содержит в себе копию файла

Жесткие ссылки: (ИСПОЛЬЗУЮТСЯ РЕДКО)
1) одинаковый inode с тем файлом, на который указывает
2) нельзя создавать для каталогов
3) выступает как новое имя для файла

СОЗДАНИЕ ССЫЛОК:
ln [опции] целевой_файл имя_ссылки
-s - для создания жесткой ссылки

ПРОСМОТР ССЫЛОК
ls -li
