#### Бонус: `copy()` vs `deepcopy()`

- `copy()` — поверхностная копия
- `deepcopy()` — рекурсивная копия всех вложенных объектов
    
```python
import copy a = [[1, 2]] 
b = copy.copy(a) 
c = copy.deepcopy(a)
```
